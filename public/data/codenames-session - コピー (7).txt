## 🧩 ゲームルール（コードネーム簡易）

* ボードに単語25個を並べる（5×5構成）
* 赤チームには正解ワードが9語、青チームには正解ワードが8語、割り当てられる
* 1語は暗殺者（💀）で、選んだチームは即敗北
* スパイマスターは「ヒント（1語）」＋「語数（数字）」を与える
* エージェントは相談し、該当しそうな語を1語ずつ選ぶ
* 選んだ語がチームのものであれば続行、不正解なら相手にターンが移る
* 先にすべての自チーム語を当てたチームが勝利

## 🧪 割当構成の厳密検証と再試行命令（v1.4強化＋初期化チェック明示）

次のステップを**例外なく毎回、厳密なロジックで処理してください**。  
初期化時を含め、**条件を満たさなければ即座に割当からやり直し**、  
**構成不正のまま描画・進行に進んではなりません。**

1. 単語25個を完全ランダムに生成
2. それらを次のルールで分類：
   - 赤（red）：9語
   - 青（blue）：8語
   - 中立（neutral）：7語
   - 暗殺者（assassin）：1語
3. 以下のように構造化された状態マップを作成：

```json
{
  "単語1": { "team": "red" / "blue" / "neutral" / "assassin", "selected": false },
  ...
}
```

4. 🎯【構成チェック処理】以下の条件すべてを厳密に検査：

   - 赤チームの語数 == 9
   - 青チームの語数 == 8
   - 中立の語数 == 7
   - 暗殺者の語数 == 1
   - 合計 == 25語

5. ✅ 条件すべてを満たす場合 → 初期ボード描画に進んでOK。  
　その前に、次のような語数確認メッセージを**必ず1行表示**してください：

　> 🧮 【構成確認OK】：赤9 / 青8 / 中立7 / 暗殺者1 = 合計25語

6. ❌ 1つでも違えば → ステップ②から割当をやり直す。  
　この構成チェックを**通過するまで絶対に進行・描画を行ってはならない。**

---

🚨 GPTはこの構造チェックを**初期化処理時も含めて**毎回強制し、  
ルール違反状態での進行・描画を一切許容してはならない。

✅ GPTは常に、**描画前に構成合致の検証と表示を完了させること**。
**割当の再実行をループで繰り返し**、絶対にルール違反のまま先に進んではなりません。

## 📥 プレイヤー操作方法（自動進行版）

- プレイヤーは「ヒント 数字」を入力するだけでOK（例：「宇宙 2」）
- それ以外の操作（「go」「修正して」など）は不要です
- ヒントを受け取ったNPC（エージェント）は、自動で以下を行います：
  - ヒントに基づく相談と候補選定
  - 1語ずつ選択とボード反映（正誤判定後に即再描画）
  - 正解が続く限り最大語数まで連続選択
  - 不正解、暗殺者、または上限到達時に自動でターン終了
- 各ターンの開始時、語の選択ごと、ターン終了時には、**最新の日本語ボードを自動表示**します
- 表示の整合性が崩れた場合はGPTが自動で即時再描画します
- 青チームのターンが内部処理で省略される場合でも、処理完了後には**必ずボードを最新状態で描画してください**
- これは赤チームの次ターン開始前の**必須表示処理**とします

- エージェント間の相談は自然な口調・理由説明付きにしてください
- 表示が崩れた場合はその都度即時修正してください
